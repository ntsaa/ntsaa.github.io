Master NTSAA in a Few Steps

Screenshot & Editing Tool - inspired by LightShot, with built-in Google Translate OCR text extraction.

1. Open the application

  • No installation required - simply extract and run NTSAA.exe directly.
  • Extraction password: ntsaa
Note:
  • If blocked by SmartScreen, choose “Run anyway” to continue.
  • If your system does not have .NET 8, Windows will automatically show the installation window (only the first time you run the app).

2. Convert image to text

  • Select the area of the image you want to recognize.
  • Press Ctrl+G or click the G icon on the toolbar.

3. Use drawing tools

  • Select the area of the image you want to edit.
  • Use drawing or text tools from the toolbar (usually located on the right side of the capture area).

4. Keyboard shortcuts

Basic:
  • Ctrl+G - Convert image to text
  • Ctrl+C - Copy image
  • Ctrl+S - Save image
  • Ctrl+X - Close
When using drawing tools:
  • Double-click mouse - Switch to full screen capture mode
  • Mouse scroll - Increase/decrease stroke thickness
  • F1 - Show/hide help
  • F2 / Ctrl+0 - Reset stroke thickness to default
  • F3 / Ctrl+- - Decrease stroke thickness
  • F4 / Ctrl++ - Increase stroke thickness
  • F8 - Toggle rounded corners
  • Hold Shift - Draw square, circle, or straight line
